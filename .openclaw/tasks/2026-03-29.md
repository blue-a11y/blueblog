## 关键结论
- HeroUI v3 **不需要 Provider**，保留纯 `layout + globals.css` 结构即可。
- HeroUI v3 **必须使用 Tailwind CSS v4**；Tailwind v3 那套教程可以直接扔了。
- 组件模式是 **compound components**，不是旧版平铺 props。
- 可访问性交互建议使用 **`onPress`**，而不是 `onClick`。
- CSS 导入顺序已经按要求固定：`@import "tailwindcss";` → `@import "@heroui/styles";`。
